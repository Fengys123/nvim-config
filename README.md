This is my nvim config written in pure lua. Support encoding rust.

# How to use?
1. Install some dependent libraries first
    - BurntSushi/ripgrep
    - sharkdp/fd
    - Nerd fonts  
2. Install nvim (version >= 0.6)
3. git clone https://github.com/nvim-lualine/lualine.nvim.git ~/.
config/nvim
4. Enjoy it!
