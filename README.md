This is my nvim config written in pure lua. Support encoding rust.

# How to use?
1. Install some dependent libs first
    - BurntSushi/ripgrep
    - sharkdp/fd
    - Nerd fonts  
2. Install nvim (version >= 0.7)
3. git clone https://github.com/Fengys123/nvim-config.git ~/.config/nvim
4. Enjoy it!
